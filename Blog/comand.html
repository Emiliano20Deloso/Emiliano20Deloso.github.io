Comandos de Terminal

En esta parte de nuestro blog podremos ver algunos de los comandos que hemos ido utilizando a lo largo de este bloque

  • ls -a: "(ls all)"" Muestra todo hasta los ocultos.
  • History: Muestra todos los comandos que hemos usado.
  • man: (nombre el comando a buscar) Te enseña el manual del comando que quieras.
  • cat: (nombre del archivo) Te muestra el contenido de un archivo.
  • mkdir: (nombre del carpeta) "Make directory" Crea un directorio.
  • touch: (nombre del archivo a crear) Crear archivos vacíos o modifica la fecha y hora de modificación.
  • alias: (nombre que queremos ponerle) (comando ya existente) Crea un alias para un comando que te quieras que se llame de una manera específica.
  • nano: (nombre del archivo a abrir) Abre un editor de texto por default en nuestra terminal.
  • vi: (nombre del archivo a editar) - vim: Otro editor de texto (aunque más complicado) que tiene la terminal.
  • rm: - vim: Otro editor de texto (aunque más complicado) que tiene la terminal.
  • cd .. : Paso atras en un archivo o carpeta
  • rm : (nombre del archivo a borrar) Borra un archivo
  • rm -r: (nombre de la carpeta a borrar) Borra una carpeta
  • rm / -fr: Comando para borrar de manera recursiva
  • mv: (nombre del archivo a mover) Otro editor de texto (aunque más complicado) que tiene la terminal.
  • rm *".cpp": (nombre del archivo a borrar) Otro editor de texto (aunque más complicado) que tiene la terminal.
  • code. : (nombre del archivo para abrirlo en un editor) Otro editor de texto (aunque más complicado) que tiene la terminal.
